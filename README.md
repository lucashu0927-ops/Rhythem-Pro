# 🎵 Rhythm Pro: Neon Beat

A fast-paced browser rhythm game where only perfect timing counts.

## 🎮 Gameplay
Press **D, F, J, K** as notes fall toward the hit line.

- Perfect → clears the note  
- Good / Bad → note stays  
- Miss → -500 score  

## 🚀 Features
- 3 difficulty levels (Level 1–3)
- Precision-based gameplay
- Real-time judgement system
- Clean neon-style interface
- Runs directly in browser (no install)

## 🕹️ Controls
| Key | Lane |
|-----|------|
| D   | Left |
| F   | Mid-left |
| J   | Mid-right |
| K   | Right |
